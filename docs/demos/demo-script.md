# Demo Script

## Intro
CloudGuard AI is an AWS-first event-driven threat detection and response platform.
It analyzes suspicious activity, assigns a risk score, and recommends or triggers a response.

## Demo Flow
1. Show project structure
2. Explain architecture at a high level
3. Run backend locally
4. Trigger a simulated suspicious event
5. Show normalized event output
6. Show calculated risk score
7. Show recommended response
8. Explain how this evolves into Sentinel AI

## Key Talking Points
- event-driven design
- modular backend
- risk scoring instead of binary logic
- future cloud security platform vision
- AWS architecture alignment

## Closing
This project demonstrates cloud architecture, security automation, and product thinking.
